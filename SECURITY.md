# Security Policy

## Supported Scope

This repository contains **reference implementations and documentation only**.
It does not run autonomously and does not manage secrets.

---

## Reporting a Vulnerability

If you discover a security issue:

- Do NOT open a public issue
- Contact the repository owner privately
- Provide reproduction steps if possible

---

## Non-Goals

This project intentionally excludes:
- Credential storage
- Key rotation
- Network perimeter enforcement

These belong in internal systems.
